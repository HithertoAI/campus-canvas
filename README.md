# campus-canvas
*A spatial social learning environment. Not an LMS. Not a forum. A place.*
# Flux Hall Academy Campus

## Core Metaphors
- **Canvas** — Infinite, but not overwhelming (you're always somewhere specific)
- **Quads** — Major areas (e.g., Sciences, Humanities, ProSocial Practice)
- **Complexes** — Specific courses, modules, or social spaces
- **Fireflies** — Visible presence indicators. Mount to travel. See where it's busy.
- **Petting Zoo** — Onboarding space. Whimsical. Learn by doing.
- **Descent** — Arrival ritual. You float down like Mary Poppins. Not "login."

## Principles
1. **Spatial presence over feeds** — Where you are matters more than what you follow.
2. **Visible activity without intrusion** — Firefly density, not notifications.
3. **Whimsy as onboarding** — Giant fireflies > tooltips.
4. **SCORM as compatibility, not identity** — We support standards, but we're not defined by them.

## Open Questions
- How does assessment work in a spatial environment?
- Can fireflies carry state (e.g., "this learner is in a quiz")?
- What happens when 100 fireflies are in one Complex? (Scalability of metaphor)
- How do you find specific content without search? (Wayfinding, not querying)

## Relationship to ProSocial Tech

- ProSocial Tech = social/community mode
- Flux Hall = learning mode
